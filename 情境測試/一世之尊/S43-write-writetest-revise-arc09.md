- ⭐**revise 收 rollup 需導演補一句**：導演 revise 指令把「不重生 _index」的例外**錯綁在「幕/POV/狀態欄變動」上**，但 rollup 是「任一同層源檔內容變＝來源 digest 變＝stale」；結果 revise 改 3 章後 `_index.ai.md` stale、系統照指令沒補。導演發現後 SendMessage 補令重生（系統順手把 ch0073 備註裡同一個手代稱也清了）。**根因＝「rollup sibling-lag」仍是常駐系統性缺口（S40 note ⑦），且這次是導演指令措辭把它觸發出來**——建議 revise skill 明文「句子層改源檔後 rollup 必連帶重生」，別靠導演逐次記得。
