# 好讀網文的跨書共通點 — A–F 基準清單

- **用途**：歸納六本公認好讀網文（詭秘之主／一世之尊／極道天魔／我有一個修仙世界／神秀之主／從紅月開始）在 A–F 六維度上的**共通做法**，作為【階段三】對照「芯片巫師」的基準清單。階段三就拿這份逐維度問「芯片巫師做到了嗎？」。
- **樣本結構**：4 本含開頭（詭秘／一世/極道/紅月，ch1–40 或 ch1–57），2 本純中段（修仙 ch794–806／神秀 ch364–461）。**同一套手法在開頭段與中段都成立**——這點很關鍵，代表這些是「常態敘事的通用技術」，不是開頭專屬花招。
- **對準的病徵**（芯片巫師的已知問題）：碎片感、生硬難讀、大量設定突然蹦出又沒解釋清楚、讀完 ch01–08 仍不清楚主角的組織／為何被催收／怎麼解決。下面每維度末尾都給「對準此病徵的可檢查判準」。

---

## 總覽：一句話共識

六本「好讀」的共同底層邏輯只有一句：**讓讀者永遠「先被一個具體情境勾住、再以剛好夠用的量獲得解釋」，而且解釋永遠貼著主角當下的動作與利害**。碎片感與生硬，本質上都是違反這條——資訊脫離情境、脫離動機、脫離畫面地「被塞入」。

---

## A. 資訊揭露與「設定如何被帶出」

**共通做法**：
1. **設定由「主角當下的動作需求」或「有私心的對話」逼出，幾乎不用旁白說明書。** — 詭秘（想看傷口→逼出投幣煤氣燈＋幣制）、一世（胖和尚玄心吹牛炫耀時漏出門派武功榜）、極道（酒坊偷聽＋城門守軍閒聊，三方印證「有鬼怪」）、修仙（虛空行走是「試」出來的，當場才補規則）、神秀（信使路上八卦帶出整套位階制度）、紅月（引子用偽史料讓讀者自行歸納世界規則）。
2. **先給現象／體驗，後給定義；名詞延後。** — 紅月最極端：ch1 先演「幻象家人」，到 ch5–6 陳菁才正式定義「污染源／精神異變者」；詭秘「愚者」牌先出現、後成代號。**體驗在前，術語在後。**
3. **一次只丟一兩個陌生詞，且每個詞出場當句就配一條「功能標籤」。** — 神秀：「九幽魔火！這神通位列三品，火焰能滅魂蝕骨……」名字＋品階＋效果＋橫向對比一句話交代完，不打斷戰鬥。
4. **硬設定塞進「有動機的問答／教學」並用對話切碎，每輪只給一小塊。** — 詭秘（鄧恩問、克萊恩答，創世神話拆成來回）、一世（玄恩上課，因主角正要修煉而「有動機聽」）、修仙（牽星老祖教學問答帶出「線／規矩」）。
5. **懸置的疑問有明確回收時點：短謎當章解、大謎養十幾章到幾十章。** — 詭秘（穿越之謎、失蹤筆記養十幾章）、極道（「這世界到底怎麼回事」壓到 ch28/33 才由角色集中揭曉）。

**對準芯片巫師病徵的可檢查判準**：
- [ ] 每個新設定第一次出現時，是否綁在一個「主角此刻正要做的動作」或「某角色說出口有私心的話」裡？還是旁白直接開講？
- [ ] 是否「先演現象再給定義」，術語有沒有延後到讀者已看過它作用之後？
- [ ] 單一場景／段落內第一次出現的陌生專有名詞，是否被控制在 1–2 個？每個是否當場配了「這是幹嘛的」一句標籤？
- [ ] 前八章拋出的疑問（主角組織是什麼、為何被催收），有沒有明確的、可指認的回收章節？還是一直懸空或一次性倒完？

## B. 具體 vs 抽象敘事

**共通做法**：
1. **關鍵轉折與「主角如何解決難題」一律具體演出，禁止「他於是成功了」跳接結果。** — 一世（丟刀換拳打舊傷，難題拆成「具體條件＋具體利用」）、詭秘（靈視開關靠主角實地推敲敲幾下）、紅月（拆怪物逐個鏡頭、ch47 麻醉槍倒數狙殺）、修仙（結嬰全程逐拍、含中途失控再脫困）、神秀（奪舍反殺搭一整個心靈世界場景演出）。
2. **「變強／獲得能力」必配代價與身體感覺，不寫成數值跳動就結束。** — 極道（作弊器升級接吐血、氣血虧空、喝參粥）、修仙（虛空行走「範圍越遠消耗翻倍」）。
3. **主角的「聰明／強」是被演示、可被讀者複述的，不是被告知的。** — 一世（每次勝利讀者能在腦內重演）、極道（識破騙子落在「化石粉的氣味」這個具體嗅覺上）。
4. **只有無關緊要的重複勞動才用時間跳接壓縮**（「轉眼六年過去」），把畫面預算集中給真正的轉折。 — 修仙、神秀都是如此。

**對準芯片巫師病徵的可檢查判準**：
- [ ] 主角「怎麼解決被催收／怎麼脫困」是逐拍演出（動作、對話、可見的招法），還是概述跳接到「他解決了」？
- [ ] 有沒有「先陷入麻煩／失控，再具體脫困」的過程，還是直接給結果？
- [ ] 能力與代價是否成對出現？讀者看得到「用了要付什麼」嗎？
- [ ] 被時間跳接壓縮掉的，是不是真的無關緊要的部分？重要轉折有沒有被誤壓縮成一句話？

## C. 場景與轉場連貫性

**共通做法**：
1. **每個場景目的單一清楚**（領任務／練功／談判／布餌等鬼／一頓飯），少有為湊字數的碎片硬拼。 — 六本一致。
2. **硬轉場先給「定位行」**：時間＋地點座標行下在場景開頭，讀者換場前先拿到座標。 — 修仙（「仙務殿，財務部的會議室之中。」「一個月之後。」）、神秀（「地點戳章」如「荒山洞府。」）、極道（「地點小標題＋分隔線」切視角）。
3. **場景之間用因果觸發串接**：新場景由上一場的一通電話／一則傳信／一個決定「推」出來，不是硬切。 — 修仙（一整章是因果滑梯：結嬰→測界域→掃到頻率→瞬移去查→回家報喜→電話引出政務）、極道（配角于姐之死是主角回府的直接引子）。
4. **大跨度時空用「一行時間橋＋角色立刻落地」補實**：跨十幾年後馬上讓一個具體角色的日常吐槽把空白填滿。 — 神秀（「匆匆十八年」後徐心兒一句吐槽落地）。
5. **多線並行用固定錨點／剪接手法縫合**：固定意象（一世的月光＋呼吸聲＝回日常）或雙線交錯剪接（紅月的主角線⇄觀測站上帝視角）。

**對準芯片巫師病徵的可檢查判準**（← 碎片感的主戰場）：
- [ ] 每個場景是否有單一明確目的？有沒有目的不明、像是拼湊的段落？
- [ ] 換場時讀者有沒有拿到「何時、何地」的座標？還是突然人就在另一個地方？
- [ ] 相鄰兩個場景之間有沒有因果連接（後一場是前一場的結果）？還是只是並列擺放、靠讀者自己腦補？
- [ ] 時間跳躍後，有沒有用一個具體角色／畫面立刻把讀者重新接住？

## D. 句子與段落可讀性

**共通做法**：
1. **短句短段為主，常一句一段**；擬聲詞／關鍵動作獨立成行做分鏡節奏。 — 極道（「呼！」獨立成行）、神秀（「然而，下一刻。」單行斷句切節奏）、紅月（短段製造「一直有事發生」的翻頁感）。
2. **對白比例高，對白承擔資訊與人物性格，敘述只做必要補位。** — 極道（胖子介紹端木婉全靠對話，敘述只補一句心理）、神秀（用『』與'…'區分說出口與心裡想）。
3. **資訊密的段落後面必接一句大白話／吐槽「洩壓」，形成緊—鬆—緊呼吸。** — 一世（設定重段後接「畫風不對啊！」）、修仙（把聖地寫成「有售後保修的壟斷企業」）。
4. **用「現代口語比喻／熟悉尺度換算」把陌生概念接住，主動對沖翻譯腔與文言僵硬。** — 極道（「一兩銀子＝一千人民幣」免背誦）、修仙（售後／保修／查賬等當代語彙）、神秀（「關我鳥事」式吐槽降門檻）、詭秘（維多利亞異世界插當代母語梗）。
5. **長句只留給氛圍／恐怖／抒情**，且用逗號切成可呼吸的節拍。 — 紅月（日常用短句，氛圍才放長句）。

**對準芯片巫師病徵的可檢查判準**（← 生硬難讀的主戰場）：
- [ ] 段落是否偏長、資訊塞滿？有沒有一句一段的節奏調劑？
- [ ] 對白比例夠不夠？資訊是靠角色「說出來」還是靠旁白「講出來」？
- [ ] 每段密資訊之後，有沒有一句白話／畫面／吐槽讓讀者換氣？還是連續高密度不喘氣？
- [ ] 陌生術語有沒有用讀者熟悉的東西（現代比喻、熟悉尺度）接住？還是硬邦邦地拋術語？
- [ ] 有沒有翻譯腔／過度修飾／長句堆疊而該用短句的地方？

> ⚠️ **D 維只到「節奏／結構」層，接不住「一讀就生硬」的字詞句法病根。** 真正下沉到「遣詞、用字、句法」（動詞強度／名詞具體度／修飾語密度／句法直接性／比喻取材／語氣顆粒／翻譯腔正面拆解）的字詞級基準，見姊妹檔 **`_字詞級共通點.md`（M1–M7）**。芯片巫師若「排版節奏其實還行、但句子就是澀」，要靠那份 M1–M7（尤其 M4 句法、M1 動詞）才定位得到——D 維查不出來。

## E. 角色／組織／動機交代

**共通做法**：
1. **主角的處境／目標／性格在極短篇幅內（常第一章一頁內）立齊。** — 一世（穿越＋被棄雜役僧＋想習武還俗，四合一在 ch1）、極道（前世老油子＋今生富家公子＋想躺平，一句話交代）、詭秘（穿越＋貧困＋想回家，前三章鎖定）。
2. **動機用「反覆的具體渴望＋對照」夯實，而非一句宣言。** — 一世（用天之驕子反襯主角的困）、極道（用「徐家滅門」把「想躺平」一擊翻轉成「必須自保」）。
3. **組織與利害衝突靠「角色的切身利害」浮現**：誰要什麼、為何為難主角，用具體事件／對話帶出，不用履歷表。 — 詭秘（值夜者線登場時，鄧恩一句「你是唯一的線索」把主角夾成嫌疑人＋怕暴露穿越）、極道（路家「八十六護衛、能外調四十人」掂出「有錢但不夠強、樹大招風」）。
4. **中段靠「對話標籤即時複述身份＋主角戰略內心獨白」持續給定位，跳讀也不掉線。** — 修仙（對話 tag 順手補「現在執掌句芒一脈的裴青霜」）、神秀（每章主角內心盤算重申「我是誰／在躲誰／下什麼棋」）。
5. **配角出場自帶一句動機**，讓讀者知道每個人為什麼站這邊。 — 修仙（孟凰兒、藍海天出場即給一句可抓的理由）。

**對準芯片巫師病徵的可檢查判準**（← 「讀完 ch01–08 仍不清楚主角組織／為何被催收」的直接對照）：
- [ ] 讀者在前 1–3 章能否清楚回答：主角要什麼？卡在哪？誰在為難他？賭注多大？
- [ ] 主角的「組織」是靠一件具體事件／一段有利害的對話立起來的嗎？還是名詞被提到但從沒被演過？
- [ ] 「為何被催收」——這個核心衝突有沒有一個像「徐家滅門」「你是唯一的線索」那樣的、具體可感的引爆點／夾擊點？
- [ ] 動機是反覆用具體渴望＋對照夯實，還是只宣告一次就指望讀者記住？
- [ ] 每個組織／勢力是靠角色的切身利害浮現，還是靠設定條列？

## F. 鉤子與張力維持

**共通做法**：
1. **章末鉤子密集且形式多樣**：驚悚鉤／抉擇鉤／喜劇鉤／反轉鉤／新資訊新道具鉤輪替，不是單調的一種。 — 詭秘（ch1 腦漿蠕動、ch40 喜劇釘子）、紅月（ch4 老式相機開下段麻煩、ch27 連環反轉）。
2. **核心張力來自「資訊差」**：或讀者與主角同步不知（同步懸念），或主角知情讀者也知情、旁人不知（優越懸念／扮豬吃虎），或讀者比主角多知（替他捏汗）。 — 神秀＋修仙的最強引擎是「讀者全知主角真身、配角全誤判」的扮豬吃虎；紅月是「讀者比主角多知道風險」的雙線剪接；一世是「主角懂無限流套路、同伴不懂」。
3. **敢先把結果給讀者，再看滿場配角猜錯**（戲劇性反諷）——張力從「會不會成功」轉為「看打臉的過程」。 — 修仙 ch798 教科書級：讀者已看到結嬰成功，整章卻讓全仙門預言他失敗。
4. **一條核心大懸念貫穿全書、故意慢燉甚至不回收**，當長線黏性。 — 紅月（「家人到底是什麼」貫穿不解）、詭秘（穿越之謎養全程）。
5. **【開頭 vs 常態是兩套節奏】**（4 本開頭書一致的發現）：
   - **開頭段**：高密度強懸念、事件驅動極強、幾乎每章一個大轉折或一次變強，用「主角命運」的未知硬拉黏性，目的是快速把讀者釘在椅子上。
   - **常態段**：轉為「養成爽感／日常經營＋輕喜劇調劑＋分批揭世界觀」，張力改由**長線大謎**在底層供能，章末鉤降級為技能解鎖／喜劇釘子。
   - **關鍵**：模仿時勿把開篇的緊繃硬套到常態段，也勿把常態的鬆弛用在開頭。**同樣是懸念，開頭懸「主角命運」，常態懸「這一場誰會死／這次怎麼打臉」。**

**對準芯片巫師病徵的可檢查判準**：
- [ ] 每章末有沒有鉤子？形式是否多樣，還是沒有／單調？
- [ ] 有沒有明確的「資訊差」引擎（誰知道什麼、誰不知道）在驅動張力？還是資訊全平攤、無落差？
- [ ] 有沒有一條貫穿的核心大懸念在底層供能？
- [ ] 開頭段的節奏是否夠緊（每章有轉折／變化），還是慢熱到讓人不清楚發生了什麼？

## 給階段三的濃縮基準（一頁版）

拿去對照芯片巫師時，優先問這 8 題（依芯片巫師病徵的殺傷力排序）：

1. **【E｜最痛】** 前八章，讀者能不能清楚回答「主角要什麼、屬於哪個組織、為何被催收、賭注多大」？這四題若答不出，就是病根。好讀書全部在前 1–3 章用**具體事件＋有利害的對話**答完這四題。
2. **【A】** 新設定是「動作需求／有私心的對話逼出＋先現象後定義＋一次1–2個詞＋當場配功能標籤」，還是「旁白說明書一次性塞入」？——這是「設定突然蹦出」的直接對照。
3. **【C】** 場景有沒有單一目的？換場有沒有時間地點座標？相鄰場景有沒有因果連接？——這是「碎片感」的直接對照。
4. **【D】** 段落是否一句一段有節奏、對白比例夠、密資訊後有洩壓、術語有熟悉比喻接住？——這是「生硬難讀」的直接對照。
5. **【B】** 主角解決難題是逐拍演出還是跳接結果？能力有沒有配代價？
6. **【A/E｜回收】** 前八章拋的疑問有沒有可指認的回收時點，還是懸空？
7. **【F】** 有沒有資訊差引擎＋章末鉤子＋一條貫穿大懸念？
8. **【F｜節奏】** 開頭段夠不夠緊（每章有轉折）？有沒有搞錯「開頭 vs 常態」的節奏檔位？
