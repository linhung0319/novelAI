# 修煉體系

這個世界的武功是真的——高手快如驚馬、無聲無息繞到你背後，再往上有金身、法身、佛祖、菩薩這種近乎神佛的層次。修煉有一條由低到高的階梯，佛門（少林）有自己一套雅稱，江湖俗稱另一套。**正文主用俗稱**，佛門雅稱當少林內部口吻的風味，由和尚嘴裡自然帶出，不當主敘述語。

## 境界階梯（由低到高）

1. **百日筑基**（佛門雅稱「定中修身」，江湖俗稱又叫「開脈」）——入定化生真氣、通經脈、填竅、強身。約百日到一年。這是**身體底子**。
2. **蓄氣**（佛門「長養聖胎」）——氣脈大通、肉身錘煉。
3. **開竅期**——練臟腑竅穴。竅開了會生出對應的感知/神通，例如「眼為肝之竅」開了眼竅≈天眼通那類，開耳竅≈天耳通。**神識感知是從這一階才有的**。
4. **外景**——能溝通天地法則，神功到這層才發真正威力。
5. 再上去：**法身／金身／道體**。金身由低到高：**羅漢金身 < 普通菩薩金身 < 大菩薩／大阿羅漢金身 < 如來金身／阿彌陀法身／菩提金身**。

> **卷一露出邊界**：上層的金身/法身/道體，卷一只當「傳說」讓人心熱、讓孟奇嚮往，正文實際進度停在**開竅期上下**，不真證到金身/法身階。上層等級表存著當背景，別在卷一鋪滿。

## 三件事要分開，別混成一根進度條（核心規則）

這是本書一個刻意設計，不是矛盾：

- **筑基（身體底子）**＝化生真氣、通經脈、填竅，把身體練成一具好胚子。
- **會武功（招式內力）**＝另一回事，要學招式、練勁路，靈魂會不會用是獨立的。
- **神識感知**＝再另一回事，要到開竅期開了對應的竅才有。

所以孟奇這具身體**已經筑基完了**（原身留下的底子，見伏筆），身體是好胚子；但**穿越來的靈魂不會用招、也沒開任何感知**，開場察覺不到背後有人（玄藏一聲佛號差點嚇死他）完全合理。三者各走各的，別當成同一條升級條。

## 真氣化生的細節（卷一只露一小段）

入定一刻鐘化生一口真氣，真氣經任督二脈行到上下顎，在那裡凝成「金津玉液」，合成一口**甘露**吞入、拿去填竅。人身有三百六十五個大竅，**每一竅要九口氣**才填滿。
> 卷一只在孟奇初學「定中修身」時露一小段：讓讀者懂什麼叫填竅、跟著感受一次甘露的清甜就好，不鋪滿全書。

## 開竅 × 丹藥的鐵律

- 開竅可借**丹藥**助力，但**同一種丹藥一生只對你有效一次**。下一個竅要換更高級、更貴的丹藥（例：天聰丸 → 天視地聽丸 → ……越開越貴）。
- **靠丹藥硬突破會留隱患**，日後大境界更難跨。門內老話：「**能自己磨就別吃藥**。」
  - **這「隱患」具體是什麼（＝上一條的現形）**：丹藥是把境界一竅一竅「硬撐」上來的，催得快、卻催得虛——**竅穴根基虛浮不穩、真氣不能一氣貫通**。平時看不出來，照樣行氣、照樣使招；可**一旦累到極處、逼到絕境，在最要命的當口，那口該接上來的真氣會接不上**。輕則當場**廢功**、真氣逆走傷身；重則那口氣直接**反噬**——人在最需要它的一瞬僵住、卸力/發勁的動作做到一半停了，**當場致命**。丹藥堆得越高、對手逼得越狠，這口氣接不上的機率越大。
  - 所以「硬吃丹藥」不只是「日後難跨大境界」的**慢性欠債**，更是一顆**埋在自己身子裡、會在生死搏殺的極限當口引爆的雷**——這才是「能自己磨就別吃藥」真正的份量。（此機制在正文首次現形＝卷二 arc05 齊正言之死；孟奇本人一路也在盤算硬吃丹藥撞開竅，日後若真走這條捷徑，就得受同一條規則約束。）
- 這條是給後續設「資源焦慮」與「捷徑代價」用的底層規則。

## 兵器／物品的分級（與現實江湖同一套標準）

兵器與物品有由低到高的分級：**普通 → 百煉 → 利器 → 寶兵 → 神兵 → 絕世神兵**。**功法不走這條軸**——功法照上面的「境界階梯」定級（筑基基礎→蓄氣→開竅→外景→法身級神功），別把功法套進兵器的級名（這是定版：功法歸境界、兵器/物品歸普通→絕世神兵，兩軸不混）。

> 🌑 **全書級大伏筆（卷一不點破）**：現實江湖與「輪回世界」（見 `任務世界引擎.md`）對**兵器／物品**用的是**同一套分級標準**（普通→…→絕世神兵），暗示兩個世界同源。這條掛全書級伏筆，卷一只讓它像個「怎麼連兩個世界的兵器標準都一樣」的小違和擺著，不解釋、不點破。（**功法不在同源線上**，走境界階梯，不是這條伏筆的線索。）

## 換取修為的代價（與任務世界引擎扣合）

在輪回世界能用「善功」兌換功法秘籍，上限可到法身級神功（動輒上萬善功），但代價分三層：
1. **善功難賺**——每一分都要拿命換（三段式：完成／殺敵／達標；量級見 `任務世界引擎.md`）。
2. **兌換一價只買到「秘籍」**——想直接把功法「上身」/入門即成，要加付「上身乘數」。**乘數隨標的稀有度／階位浮動**：爛大街、低階的 **×2**；越高階、越稀有的越難硬塞進身體、越趨近 **×5**。孟奇卷一碰得到的多在 **×2–3**。（強化「別暴富速通」「硬塞高階留隱患」的張力。）
3. **想靠丹藥硬突破**省時間，就吃上面「開竅×丹藥」那條的隱患——不只大境界更難跨，還埋下「極限對抗時真氣接不上而反噬（輕則廢功、重則當場致命）」的雷（見上）。

（兌換機制的完整規則在 `任務世界引擎.md`；此處只記「換修為要付什麼代價」這一面。）
