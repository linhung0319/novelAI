# 角色塑造

> **用途**：本檔承載「方法」，收錄角色的**塑造／呈現**手法（怎麼把一個角色寫得立體、可信、有記憶點）。供 `develop`／`organize` 角色打底、`beat-sheet` 讓幕推進角色形象、`write`／`revise` 把角色寫進正文引用。
> **與 `需求四象限.md` 的分工**：`需求四象限.md` 管角色的**需求與弧線**（他要什麼、怎麼變）；本檔管角色的**塑造與呈現**（他是什麼樣的人、怎麼讓讀者信）。角色的正/平/負弧線與圓扁分類在 `需求四象限.md`。
> **核心信念**：「動作即人物」——「比起在故事中直述『張先生是個心機重的變態』，用反差安排事件、控制真相揭露，反而更能讓角色立體」（【人物】L404）。與 `展示而非講述與五感描寫.md` 同源。

## 一、三面向分析法（角色的完整座標）

出自埃格里《戲劇寫作的藝術》（【人物】L197）。一個角色須由三面構築才完整：

| 面向 | 內容 |
|---|---|
| **生理面** | 性別、年齡、種族、身形、面貌、健康、缺陷、口音…——即便鼻頭高一公分，成長中也可能滾成截然不同的角色（L206-221） |
| **社會面** | 生活背景：在哪長大、家庭父母、學校、工作如何塑造其政治/宗教/金錢/人生觀——「沒有過去的角色，走不出踏實的未來」（L230-254） |
| **心理面** | 多由生理＋社會構成（也有與生俱來的智商/性向/敏感度）：執著、壓抑、畏懼、自大、自卑…（L260-266） |

> 洛克偏好：比起寫「角色履歷表」再推想行為，他更愛直接寫下**角色的行為記事**（且讓這些行為都出現在故事中），更直觀精準（L275）。反對花太多時間寫效率低的履歷表。

## 二、三層行為模式與內外反差（動作即人物）

「越私密的行為越能代表真正的性格」（【人物】L299）。三層（依親疏）：**社交行為**（客套/陌生時的反應）→ **親近行為**（知心對象面前）→ **私密行為**（四下無人時，最真實的自己）。

**關鍵應用——刻意設計內外反差**（L362-389）：真實人生三層差異不大，但小說角色要「刻意」製造反差。做法：**先「由內」設定好真實的樣貌，再設置「外在」落差的樣貌**。
- 例：古道熱腸（私密）→ 外冷內熱（親近）→ 非常冷漠（社交）。
- **配角**只需顯露社交、頂多到親近（最好有反差）；**主角/要角**要多一層私密，一樣要有反差。
- 反差＝「虛假與真實」的戲劇效果，讓讀者到最後才驚呼角色真相。

**真相揭露的時機決定效果**（L395，呼應 `戲劇張力與懸念.md`「隱瞞」）：主角不知、讀者先知＝為主角著急；主角與讀者最後才一起知＝驚訝；主角知、讀者不知＝賣關子、製造好奇。

## 三、由外而內塑造三階段（形象→邏輯→情緒）

像洋蔥分層（【人物】L410-569）：

1. **形象（表面、直觀、引人注意）**：先聲奪人＝**打破刻板印象**——肥宅武功高強、殺手不殺人、少林弟子想踢足球。好形象不複雜但很吸睛，先做好再讓讀者深入。
2. **邏輯（有理有據、禁得起推敲）**：建立角色的內在世界，三細項——**自我意識**（把角色丟進新事件，你能「直覺」想出他會怎麼做＝「角色會自己活了」）、**合理反應**（帶入角色做選擇後複查：這角色真會這樣嗎？有無違背個性/與前文矛盾/過或不及？）、**利害關係**（角色間是敵是友、是愛是恨，行為才有跡可循、產生張力——「哥哥與妹妹相愛」一句就讓人好奇結局）。
3. **情緒（最核心，能顛覆邏輯、超越表象）**：給角色一個**異常強烈的核心情緒**（驕傲/忌妒/貪婪/癡戀/惡毒皆可），讓他成為「某種情緒的化身」——當角色能代表一種情緒時，行為就算有違常理讀者也能接受。

> 三者應吻合：「誠於中，形於外」；若形象與內在悖離（和藹的人卻有強烈憤恨），也必須有一個「為什麼」讓表裡不一說得通——這種矛盾反而讓角色更突出（L563-569）。

## 四、塑造六法（由淺至深取信讀者）

了解一個人的六種途徑，也是角色取信讀者、由淺至深的順序（【人物】L1682-1799…）：

| 法 | 做法 | 深淺 |
|---|---|---|
| 直述法 | 直接說「他是善良的人」 | 快而不明、淺而不實；後面須接行為/事蹟佐證（除非刻意營造虛假/疏離感） |
| 事蹟法 | 由他人談論、過往事蹟佐證（三人成虎、誇張事蹟；出場前先賣關子抬轎） | 由昔知今、由事塑形，比自己說更可信 |
| 象徵法 | 外觀物品／行為的刻板印象（十字架＝虔誠、掉出手槍＝危險） | 借象塑形、見微知著，一亮相就被清楚認識 |
| 對話法 | 話如其人、什麼人講什麼話（見 `對白.md`） | 但言語可真可假 |
| 行動法 | 觀察本人的行為（見 `展示而非講述與五感描寫.md`） | 比言語更可信 |
| 內心法 | 洞悉角色心聲（見 `視角與敘事人稱.md` Deep POV） | 最深 |

## 五、更多方法（指路）

以下方法本檔不展開，需要時取 `Data/【人物】.txt` 原文：
- **人物命名 5 招**（【人物】L1）：用名字暗示性格/身分/命運。
- **FLBW 角色壓力設計法**（【人物】L578）：用壓力情境逼出角色性格。
- **8 種角色原型（英雄之旅）**（【人物】L1844 起）：英雄/導師/守門者/使者/變形者/陰影/盟友/搗蛋鬼——與 `結構公式.md`「編劇九階段」的英雄之旅對應，排幕時可參照。
- **臨摹與混搭**（【人物】L3264）：從既有角色臨摹、再混搭出新角色。
- **圓形/扁平人物、正/平/負弧線**：見 `需求四象限.md` 第五節。

## 供哪些 skill 參照

- `develop`／`organize`：角色打底時用三面向盤點、由外而內定核心情緒、給重要配角至少一層反差（呼應 develop「角色需求 stub」最低要求）。
- `beat-sheet`：排幕時安排情節逐步揭露三層行為、消除或揭穿內外落差。
- `write`／`revise`：把角色寫進正文時優先用塑造六法的中後段（象徵/對話/行動/內心）而非直述法；反差與真相揭露時機交由 POV 與懸念控制（見 `視角與敘事人稱.md`、`戲劇張力與懸念.md`）。
