# 角色塑造

> **用途**：本檔承載「方法」，收錄角色的**塑造／呈現**手法（怎麼把一個角色寫得立體、可信、有記憶點）。供 `develop`／`organize` 角色打底、`beat-sheet` 讓幕推進角色形象、`write`／`revise` 把角色寫進正文引用。
> **與 `需求四象限.md` 的分工**：`需求四象限.md` 管角色的**需求與弧線**（他要什麼、怎麼變）；本檔管角色的**塑造與呈現**（他是什麼樣的人、怎麼讓讀者信）。角色的正/平/負弧線與圓扁分類在 `需求四象限.md`。
> **核心信念**：「動作即人物」——「比起在故事中直述『張先生是個心機重的變態』，用反差安排事件、控制真相揭露，反而更能讓角色立體」（【人物】L404）。與 `展示而非講述與五感描寫.md` 同源。

## 一、三面向分析法（角色的完整座標）

出自埃格里《戲劇寫作的藝術》（【人物】L197）。一個角色須由三面構築才完整：

| 面向 | 內容 |
|---|---|
| **生理面** | 性別、年齡、種族、身形、面貌、健康、缺陷、口音…——即便鼻頭高一公分，成長中也可能滾成截然不同的角色（L206-221） |
| **社會面** | 生活背景：在哪長大、家庭父母、學校、工作如何塑造其政治/宗教/金錢/人生觀——「沒有過去的角色，走不出踏實的未來」（L230-254） |
| **心理面** | 多由生理＋社會構成（也有與生俱來的智商/性向/敏感度）：執著、壓抑、畏懼、自大、自卑…（L260-266） |

> 洛克偏好：比起寫「角色履歷表」再推想行為，他更愛直接寫下**角色的行為記事**（且讓這些行為都出現在故事中），更直觀精準（L275）。反對花太多時間寫效率低的履歷表。

## 二、三層行為模式與內外反差（動作即人物）

「越私密的行為越能代表真正的性格」（【人物】L299）。三層（依親疏）：**社交行為**（客套/陌生時的反應）→ **親近行為**（知心對象面前）→ **私密行為**（四下無人時，最真實的自己）。

**關鍵應用——刻意設計內外反差**（L362-389）：真實人生三層差異不大，但小說角色要「刻意」製造反差。做法：**先「由內」設定好真實的樣貌，再設置「外在」落差的樣貌**。
- 例：古道熱腸（私密）→ 外冷內熱（親近）→ 非常冷漠（社交）。
- **配角**只需顯露社交、頂多到親近（最好有反差）；**主角/要角**要多一層私密，一樣要有反差。
- 反差＝「虛假與真實」的戲劇效果，讓讀者到最後才驚呼角色真相。

**真相揭露的時機決定效果**（L395，呼應 `戲劇張力與懸念.md`「隱瞞」）：主角不知、讀者先知＝為主角著急；主角與讀者最後才一起知＝驚訝；主角知、讀者不知＝賣關子、製造好奇。

## 三、由外而內塑造三階段（形象→邏輯→情緒）

像洋蔥分層（【人物】L410-569）：

1. **形象（表面、直觀、引人注意）**：先聲奪人＝**打破刻板印象**——肥宅武功高強、殺手不殺人、少林弟子想踢足球。好形象不複雜但很吸睛，先做好再讓讀者深入。
2. **邏輯（有理有據、禁得起推敲）**：建立角色的內在世界，三細項——**自我意識**（把角色丟進新事件，你能「直覺」想出他會怎麼做＝「角色會自己活了」）、**合理反應**（帶入角色做選擇後複查：這角色真會這樣嗎？有無違背個性/與前文矛盾/過或不及？）、**利害關係**（角色間是敵是友、是愛是恨，行為才有跡可循、產生張力——「哥哥與妹妹相愛」一句就讓人好奇結局）。
3. **情緒（最核心，能顛覆邏輯、超越表象）**：給角色一個**異常強烈的核心情緒**（驕傲/忌妒/貪婪/癡戀/惡毒皆可），讓他成為「某種情緒的化身」——當角色能代表一種情緒時，行為就算有違常理讀者也能接受。

> 三者應吻合：「誠於中，形於外」；若形象與內在悖離（和藹的人卻有強烈憤恨），也必須有一個「為什麼」讓表裡不一說得通——這種矛盾反而讓角色更突出（L563-569）。

## 四、塑造六法（由淺至深取信讀者）

了解一個人的六種途徑，也是角色取信讀者、由淺至深的順序（【人物】L1682-1799…）：

| 法 | 做法 | 深淺 |
|---|---|---|
| 直述法 | 直接說「他是善良的人」 | 快而不明、淺而不實；後面須接行為/事蹟佐證（除非刻意營造虛假/疏離感） |
| 事蹟法 | 由他人談論、過往事蹟佐證（三人成虎、誇張事蹟；出場前先賣關子抬轎） | 由昔知今、由事塑形，比自己說更可信 |
| 象徵法 | 外觀物品／行為的刻板印象（十字架＝虔誠、掉出手槍＝危險） | 借象塑形、見微知著，一亮相就被清楚認識 |
| 對話法 | 話如其人、什麼人講什麼話（見 `對白.md`） | 但言語可真可假 |
| 行動法 | 觀察本人的行為（見 `展示而非講述與五感描寫.md`） | 比言語更可信 |
| 內心法 | 洞悉角色心聲（見 `敘事距離與DeepPOV.md` Deep POV） | 最深 |

## 五、人物命名（三要領＋三提醒）

「怎麼幫人物取名」是最重要又最不重要的問題（`Data/【人物】.txt` L1-185）。三大要領：

1. **別讓讀者出戲**（最重要）：不違背故事背景（古代小說叫「志玲」）、不違背人物形象（絕世美女叫「阿狗」）、念起來不拗口。**帥哥配蠢名、現代配矯情名（最常見）、古代配現代名都是出戲大忌**——太夢幻/帥氣的名字反而出戲。
2. **名字與人物意涵連結**（可遇不可求）：金庸是高手——岳不群語出「小人合而不群」、袁承志承袁崇煥遺志。有靈感就用，別鑽牛角尖。
3. **進階考慮音與義**：聲韻上避免連續同聲母/韻母/同調、結尾多用二/四聲較發散有力（金庸六位主角四聲結尾＝堅定感）；字源上留意姓氏出處與時代/身分匹配。

> **三個額外提醒**：好記（平易近人，如「阿拓／阿克」）、好看（別用字型相似字混淆角色）、不撞名（別撞諧星或負面人物）。取不出來時的湊名法：朋友名→換個百家姓→翻字典換一字。**但別忘了：「超棒的好名字配上超爛的故事＝超爛的故事」，名字只是錦上添花。**

## 六、FLBW 角色壓力設計法（設定「不足」＝設定主線）

主角若十全十美就沒有成長空間；創造角色時要留「不足之處」讓他成長改變。Serita Deborah Stevens 歸納角色成長的四個類型「FLBW」（`Data/【人物】.txt` L578-678）：

| 類型 | 內涵 | 範例 |
|---|---|---|
| **恐懼 Fear** | 與生俱來或個人獨有的恐懼物，讓他在故事中克服 | 《王者之聲》口吃國王必須公開演說 |
| **極限 Limit** | 「求而不得」——想要更多錢/能力/健康卻得不到，逼他突破自己 | 《大亨小傳》蓋茲比千方百計仍換來心碎 |
| **阻礙 Block** | 最直觀、且是**外在**約束（他人/自然/社會），主角與之對抗排除 | 《刺激1995》的監獄 |
| **傷痕 Wound** | 過往傷痛造成的內心陰影，讓角色在類似情境逃避退縮，故事要他從逃避到面對 | 《全面啟動》走不出害死前妻的陰影 |

> **關鍵**：「決定了不足，也幾乎決定了故事主線」——故事即主角挑戰彌補不足的過程。狡猾的編劇會同時設兩三種、讓內外因素環環相扣增加壓力與張力。（FLBW 與 `需求四象限.md`「落差/需要」、`結構公式.md`「阻礙與挫折」同源——都是「先立不足，再讓角色掙來改變」。）

## 七、八種角色原型（英雄之旅）

辨識角色原型有助解析與提高編劇效率（`Data/【人物】.txt`「懶人版－小說的8種角色原型」L1844-2027）。八種原型與其**戲劇功能**：

| 原型 | 戲劇功能 |
|---|---|
| **英雄** | 成長最多、領悟最多、突破最多；三標誌：成長／主動行動／為他人或理想犧牲。是讀者的象徵 |
| **師傅** | 教誨／推動／給予，最終為讓英雄成長（常在開場給資訊、關鍵時當頭棒喝） |
| **使者** | ＝**動機**：提供動機、強迫上路，讓故事有前進方向 |
| **守門者** | ＝**試煉**：踏上旅程前的基礎測試（跨越點），逼英雄檢視內在 |
| **變形者** | ＝**懸念與驚奇**：立場不斷變化（一下幫一下害），但須合理；扛起營造謎團的重任 |
| **陰影** | ＝**阻礙**：對英雄目標造成阻礙的一切；外部陰影常由內心陰影順勢而生（外→內→外） |
| **盟友** | ＝**更多元的情感連結**：協助英雄、豐富情感、與英雄互補 |
| **搗蛋鬼** | ＝**放鬆、醞釀**：穿插歡樂平衡情緒，最後把歡笑化成感動 |

> **與結構的對應**（呼應 `結構公式.md`「編劇九階段」英雄之旅）：師傅/守門者/使者象徵開場，盟友/變形者/陰影接力形成中段，揭曉變形者/解決陰影是結尾，搗蛋鬼穿插全場。**原型不綁死角色**——可兼任、變換，甚至不是人（天氣、心情）。

## 八、履歷表爭議：記事為主、資料為輔

洛克**反對「冗長而無用」的角色履歷**（`Data/【人物】.txt`「我反對寫角色履歷表的2大原因」L2884-3000、「資料氾濫4弊＆履歷5要素」L3006-3198）。這是第一節「偏好行為記事勝過履歷表」的完整版。

**反對的兩大原因**：① **邊際效益遞減**——1000 字掌握 80%、5000 字 90%、25000 字才 95%，越花時間 CP 值越低；② **「豐富資料」≠「生動角色」**——你很了解角色，卻未必**傳達給讀者**（故事寫的是角色的「現在與未來」，不是「過去資料」）。

**核心洞見**：
> **資訊只是暗示，情節才是明說。**「壯漢拳擊手」暗示衝動暴力、「瘦小四眼仔」暗示懦弱——但角色在**情節下鮮明強烈的反應**才真正塑造特色（「過去的行為是角色未來的判斷」）。

**資料氾濫的四弊**：時間成本難取甜蜜點／作者認知難有效傳給讀者／無意義表面資料無助了解角色／資料對讀者的吸引力弱於情節。

**有效率的角色履歷「5 要素」**（其餘外貌背景等發想前先寫這五項，方向才明確）：
1. 角色身分、目標簡述；
2. 性格簡述、中心思想；
3. 少數外觀／行為特點（少即是多——只記最顯眼的，如「一頭亮紅頭髮」）；
4. 在故事中的存在意義、戲劇效果、劇情任務；
5. 彰顯角色觀念、預備呈現在故事中的經歷與言行。

> 這正是本系統 `設定/角色/<名>.md`（源，自由手寫）只要求「角色需求 stub」最低要求、不逼寫大部頭設定集的方法論依據——把力量用在「暗示＋情節反應」而非堆資料。

## 九、臨摹與混搭（角色創造的秘技）

「**天下無原創**」——喬治・普羅蒂《三十六劇》整理了 36 種劇情模式，人性千年難變，只要主角是人就跑不出其延伸（`Data/【人物】.txt`「角色創造的秘技：臨摹與混搭」L3264-3449）。所謂原創「只是看不出痕跡的、在容許範圍內的臨摹」。

- **臨摹型**：從讀過的作品中找一個角色，**只抓取他的個性**（善良/衝動/優柔寡斷…）注入新角色——不抄情節、不抄設定。原理：「**角色形成與劇情／行為連動**」「**性格由行為展現，行為由讀者解讀**」——用演不用說，經歷不同事件後，讀者腦中「阿彥是阿彥、哈利是哈利」。（抄個性沒人發現，抄情節設定＝百分百抄襲。）
- **混搭型**：挑兩三個早活在你心中的角色融合（韋小寶的無賴＋呂布的武藝；相近的混出小變化、反差大的混出新火花），再用「行為記事」寫下融合角色的各項反應。

> **底層原理**：「塑造角色說穿了，只是創造一個**判斷邏輯**，而判斷邏輯由過去經歷的事件形成」——臨摹/混搭直接借用原作累積出的成熟邏輯，快速又生動。這與第三節「自我意識（角色會自己活）」、第八節「記事為主」是同一套「角色＝可預測的判斷邏輯」觀。

## 供哪些 skill 參照

- `develop`／`organize`：角色打底時用三面向盤點、由外而內定核心情緒、給重要配角至少一層反差（呼應 develop「角色需求 stub」最低要求）。
- `beat-sheet`：排幕時安排情節逐步揭露三層行為、消除或揭穿內外落差。
- `write`／`revise`：把角色寫進正文時優先用塑造六法的中後段（象徵/對話/行動/內心）而非直述法；反差與真相揭露時機交由 POV 與懸念控制（見 `視角與敘事人稱.md`、`戲劇張力與懸念.md`）。
