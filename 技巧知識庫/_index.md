# 技巧知識庫索引

這份索引是知識庫的**檢索入口**。技法檔彼此是不同方法、扁平並列（不用資料夾分類——技法與 skill 是多對多，資料夾會強迫單一歸屬）；「層」欄只是輕量標籤，不是目錄。

**兩種用法：**
1. **臨場查找（discovery）**：沒有 skill 觸發、但要回答寫作技巧問題時（見 `CLAUDE.md`「回答任何寫作技巧問題前，先查技巧知識庫」），先讀本表的「何時查它」欄，定位到對的檔，再讀該檔對應小節。
2. **維護指針**：各 SKILL.md 用寫死的路徑＋章節錨點直達知識檔；改動技法檔時，對照「被哪些 skill 引用」欄回頭檢查指針。
